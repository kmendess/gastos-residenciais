# Database

Esta pasta contém os scripts SQL utilizados para criação e versionamento da estrutura do banco de dados.

## Organização

Os scripts seguem um padrão sequencial para facilitar a execução e controle de alterações no banco.

Exemplo:
0001_create_table_pessoa.sql
0002_create_table_categoria.sql
0003_create_table_transacao.sql

## Objetivo

Permitir a recriação do banco de dados em qualquer ambiente de forma controlada e versionada.
